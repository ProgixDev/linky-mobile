# Glossary

| Term                 | Meaning here                                                                                                   |
| -------------------- | -------------------------------------------------------------------------------------------------------------- |
| **Agent**            | A coding AI (Claude Code, Cursor, Codex…) operating in this repo under AGENTS.md                               |
| **Harness**          | Everything that injects context into agents and rejects bad output: docs tree, lint rules, tests, personas, CI |
| **Feature (slice)**  | A folder in `src/features/` owning one product capability end-to-end                                           |
| **Public API**       | A feature's `index.ts` — the only sanctioned import surface                                                    |
| **CUJ**              | Critical User Journey — a flow that must never break (docs/quality)                                            |
| **CNG**              | Continuous Native Generation — `ios/`/`android/` generated by prebuild, never committed                        |
| **Fingerprint**      | Hash of the native runtime; decides OTA update vs store build                                                  |
| **OTA / EAS Update** | Shipping JS+assets directly to devices without a store release                                                 |
| **Dev client**       | Custom development build (`expo-dev-client`) replacing Expo Go                                                 |
| **Painted door**     | Real UI + no-op backend, instrumented to measure demand before building                                        |
| **Exec plan**        | Phased implementation plan in docs/architecture/exec-plans                                                     |
| **ADR**              | Architecture Decision Record                                                                                   |
| **Persona**          | A reviewer lens (docs/personas) run by CI as an AI subagent                                                    |
| **Proof of work**    | Evidence in a PR that the change works: tests, screenshots, Argent session                                     |
| **Argent**           | Software Mansion's agentic toolkit (MCP) for driving the iOS Simulator/Android emulator                        |
| **Maestro**          | YAML-based deterministic mobile E2E runner                                                                     |
| **Ship report**      | AI-written summary PDF generated on every push to main                                                         |
